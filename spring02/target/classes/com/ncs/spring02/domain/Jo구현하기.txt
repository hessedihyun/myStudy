** Jo 관리 시스템 **
=> CRUD 구현
=> Java: JoDTO, JoDAO, JoService, JoController (클래스 파일)
=> Jsp: joList.jsp, joDetail.jsp, joInsert.jsp, joUpdate.jsp
=> home: 메뉴 <a href="jo/joList">JList</a>
=> joList에서 jno를 클릭하면 joDetail 출력

> 선생님 방식
상단에 MemberService mservice;
@AllArgsConstructer 있으면 @Autowired 필요 없음(어차피 lombok 사용시)
=> 모든 멤버변수를 초기화하는 생성자 자동 추가 & 사용
=> 그러므로 아래의 @Autowired 생략 가능

if("U".equals(jCode)) uri = "jo/joUpdate";
if("D".equals(jCode)) model.addAttribute("banana", mservice.selectJoList(dto.getJno()));

DAO에서 오류가 나면 Service에도 빨간 불이 뜨는데, Controller에서는 안뜸.
=> Service가 Controller와 DAO 사이의 결합도를 낮춰준다는 뜻
